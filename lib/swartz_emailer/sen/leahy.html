Mail Delay

Postal mail delivered to United States Senate offices undergoes an extensive screening process for security reasons, causing a delay in delivery. For more expeditious correspondence, please use the contact form on this page. If you need to send supporting materials, please fax them to (202) 224-3479. For urgent matters, please call (202) 224-4242.
